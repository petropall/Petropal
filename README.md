# ⛽ PetroPal — On-Demand Fuel Delivery Website

A complete, production-ready static website for **PetroPal**, an on-demand fuel delivery service.

## 🚀 Features

- **Hero Section** — Animated fuel delivery tracker card, stats, CTA buttons
- **Services** — Petrol, Diesel, Gasoline, Emergency Rescue, Scheduled Refueling, Fleet/Corporate
- **How It Works** — 4-step visual process
- **Order Form** — Full order form with GPS location, fuel type, delivery type selection
- **Pricing** — Standard, Pro (₹599/mo), and Fleet plans
- **Testimonials** — Customer reviews
- **About** — Company story, values, founding info
- **App Download Banner** — iOS & Android app promotion
- **Contact** — Contact form + info + social links
- **FAQ** — Accordion-style frequently asked questions
- **Footer** — Full sitemap footer

## 📦 Project Structure

## 🌐 Deploy to GitHub Pages

1. Create a new GitHub repository named `petropal`
2. Upload all files (maintain folder structure)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   `https://yourusername.github.io/petropal/`

## 🎨 Tech Stack

- Pure HTML5, CSS3, Vanilla JavaScript
- Google Fonts: Bebas Neue, DM Sans, Space Mono
- Font Awesome 6.5 icons
- No frameworks, no build tools — open index.html directly!
